# Document Types — Panduan per Jenis Dokumen Polytron

## 1. PRESENTASI (PPTX)

### Slide Cover (Halaman Pertama)
```
ELEMEN WAJIB:
[ ] Logo Polytron — kiri atas (white atau full color)
[ ] Judul besar — tengah atau kiri, 2 baris, kata kunci di-highlight kuning/merah
[ ] Tagline/subtitle — di bawah judul, lebih kecil, abu
[ ] Label unit/divisi — kanan atas atau kiri bawah
[ ] Footer bar merah — paling bawah
[ ] URL www.polytron.co.id — kiri footer
[ ] Nama divisi — kanan footer
[ ] P-mark ikon — kanan bawah sebagai dekoratif (opsional)

BACKGROUND:
- Putih bersih (#FFFFFF) — standar
- Abu sangat terang (#F5F5F5) — alternatif
- Foto dengan overlay gelap — untuk kesan dramatis

WARNA JUDUL:
- Hitam untuk kata deskriptif
- Merah atau kuning untuk kata kunci (bold + highlight)
```

### Slide Konten Standar
```
LAYOUT OPSI A (Teks + Visual):
| [Judul Slide] |
|----------------|--------------|
|  Visual/       |   Teks/      |
|  Diagram       |   Bullet     |
|  (40–50%)      |   (50–60%)   |
| [Logo kecil] [Footer Merah] |

LAYOUT OPSI B (Full Text dengan Ikon):
| [Judul Slide]                |
|------------------------------|
| 🔴 Poin 1 — [teks]          |
| 🔴 Poin 2 — [teks]          |
| 🟡 Callout box               |
| [Logo kecil] [Footer Merah] |

LAYOUT OPSI C (Data/Tabel):
| [Judul Slide]                |
|------------------------------|
| [Tabel dengan header merah] |
| [Catatan/keterangan bawah]  |
| [Logo kecil] [Footer Merah] |
```

### Slide Closing
```
BACKGROUND: Merah penuh (#E8002D) — WAJIB
TEKS: Putih, Bold, Besar
KONTEN:
  - Tagline motivasional (baris 1, besar)
  - Sub-teks (baris 2, lebih kecil)
  - Logo Polytron versi putih
```

### Slide Transisi / Section Divider
```
- Background hitam atau abu gelap dengan teks putih
- Atau background merah dengan angka section besar
- Judul section di tengah, Bold
```

## 2. DOKUMEN WORD (DOCX)

### Struktur Halaman Pertama
```
[ Header ]
  - Logo Polytron kiri atas
  - Judul dokumen center (Bold, 18–20pt, merah atau hitam)
  - Kode dokumen / versi kanan atas
  - Garis pemisah merah tebal di bawah header

[ Body ]
  - Margin sesuai spesifikasi
  - Konten dimulai setelah garis

[ Footer ]
  - www.polytron.co.id | [Nama Dokumen] | Halaman X dari Y
```

### Heading Styles
```
Heading 1: Montserrat Bold, 14pt, MERAH (#E8002D), spacing before 12pt
Heading 2: Montserrat SemiBold, 12pt, Hitam (#1A1A1A), spacing before 8pt
Heading 3: Montserrat Medium, 11pt, Charcoal (#2C2C2C), italic opsional
Body: Open Sans Regular, 10.5pt, Dark Gray (#4A4A4A), line spacing 1.15
```

### Tabel dalam Dokumen
```
Header: Background merah (#E8002D), teks putih Bold 10pt
Row genap: #F9F9F9
Row ganjil: #FFFFFF
Border: 0.5pt solid #DDDDDD
Padding cell: 0.2cm atas-bawah, 0.4cm kiri-kanan
```

### Callout Box dalam Dokumen
```
Buat dengan text box atau shape:
- Kritis: background #E8002D, teks putih
- Penting: background #FFF9C4 (kuning muda), border kiri 4pt #F5C518
- Info: background #F5F5F5, border kiri 4pt #9E9E9E
```

## 3. SPREADSHEET / MATRIKS (XLSX)

### Sheet Matriks Kompetensi
```
Header baris: Background #E8002D, teks putih, bold
Header kolom: Background #F5C518, teks hitam, bold
Data row ganjil: #FFFFFF
Data row genap: #F5F5F5
Gap/negatif: teks merah (#CC0000), background #FFEBEE
Gap/nol/positif: teks hijau (#2E7D32), background #E8F5E9
```

### Sheet Kompensasi / Skala Gaji
```
Header: Background #1A1A1A, teks putih
Sub-header: Background #4A4A4A, teks putih
Data: Background putih, border #DDDDDD
Currency/angka: rata kanan, format "Rp #,###"
Highlight: Background #FFF9C4 untuk sel yang perlu perhatian
```

## 4. EMAIL RESMI

### Format Email HR ke Karyawan
```
Subject: [KATEGORI] Judul yang Jelas dan Informatif
  Kategori: INFO | PENTING | PERLU TINDAKAN | DEADLINE: [tanggal]

Body:
  Paragraf 1: Konteks — apa dan mengapa
  Paragraf 2–3: Isi utama — detail, ketentuan, atau perubahan
  Paragraf 4: Tindakan yang diperlukan (jika ada)
  Closing: Kontak untuk pertanyaan

Signature:
  [Nama Lengkap]
  [Jabatan]
  Human Resources — Polytron
  [No. HP/Ext]
  www.polytron.co.id
```

### Format Email HR ke Manajemen/Direksi
```
Subject: [BRIEF] Judul Ringkas — Untuk Persetujuan/Informasi

Body:
  1 paragraf: Executive summary
  Bullet points: Poin-poin kunci
  1 kalimat: Tindakan yang diharapkan + deadline

Signature: sama seperti di atas
```

## 5. INFOGRAFIS DAN ONE-PAGER

### Struktur One-Pager HR
```
[ HEADER ]
  Logo + Judul + Tagline
  Background: Merah atau Hitam, teks putih

[ SECTION 1 — Highlight Numbers ]
  3–4 angka besar dengan label
  Background: Kuning atau Abu terang

[ SECTION 2 — Konten Utama ]
  Teks dengan ikon atau diagram kecil
  2–3 kolom

[ SECTION 3 — Callout/Quote ]
  Kutipan atau statement penting
  Background: Merah, teks putih

[ FOOTER ]
  Logo + URL + Kontak
```

### Infografis Proses
```
Gunakan diagram flowchart horizontal atau vertikal
Warna node:
  - Start/End: Background kuning (#F5C518)
  - Proses: Background kuning (#F5C518), border charcoal
  - Keputusan (diamond): Background kuning, border charcoal
  - Arrow: Charcoal (#2C2C2C), ketebalan 1.5pt
  - Label "Ya/Tidak": Font kecil, charcoal
```

## 6. PANDUAN KONTEN PER UNIT HR

### HR Learning & Development
```
Warna dominan: Kuning + Hitam
Tone: Edukatif, memberdayakan, growth-oriented
Visual: Diagram matriks, flow kompetensi, foto training
Statement khas: "Wujudkan Potensi Terbaik"
Struktur slide: Banyak diagram dan tabel matriks
```

### Compensation & Benefits
```
Warna dominan: Merah + Putih
Tone: Aspiratif, transparan, benefit-focused
Visual: Ikon benefit, foto profesional, tabel comparison
Statement khas: "Your Career • Your Reward • Your Future"
Struktur slide: Feature cards dengan ikon, tabel tiered benefit
```

### Good Corporate Governance (GCG)
```
Warna dominan: Merah kuat + Putih
Tone: Tegas, autoritatif, zero-tolerance
Visual: Minimal, clean, ikon sederhana
Statement khas: "Integrity is at our core."
Closing slide: Merah penuh, teks putih, statement kuat
```

### Organization Development
```
Warna dominan: Kuning + Hitam
Tone: Strategis, sistematis, forward-looking
Visual: Framework diagram, pilar/kolom, matrix
Statement khas: "Strategy builds structure, People drive growth."
Struktur slide: Heavy diagram, framework visualization
```

### Whistleblowing System
```
Warna dominan: Merah kuat
Tone: Serius, protektif, empowering (encourage reporting)
Visual: Minimal, beberapa foto konseptual
Statement khas: "Reporting is an act of loyalty, not disloyalty."
Tone khusus: Jangan menakut-nakuti — justru encourage dan reassure
```

### Recruitment / Employer Branding
```
Warna dominan: Merah + Kuning + Putih (semua elemen brand)
Tone: Inspiratif, welcoming, aspiratif
Visual: Foto tim, suasana kerja, foto produk
Statement khas: "Join a company where you are not just an expense, but an asset."
```

## 7. CHECKLIST QUALITY ASSURANCE

### Sebelum Final PPTX
```
[ ] Cover: logo, judul, footer
[ ] Setiap slide punya footer merah
[ ] Font konsisten (maks. 2 font)
[ ] Warna sesuai palet unit
[ ] Tidak ada teks overflow/terpotong
[ ] Gambar resolusi baik (tidak pixelated)
[ ] Semua teks terbaca (kontras cukup)
[ ] URL di setiap footer
[ ] Tidak ada typo / inkonsistensi terminologi
[ ] Closing slide ada dan kuat
```

### Sebelum Final DOCX
```
[ ] Header: logo + judul + kode dokumen
[ ] Garis merah pemisah di bawah header
[ ] Heading hierarchy konsisten
[ ] Footer: URL + nama dokumen + nomor halaman
[ ] Tabel: header merah + formatting konsisten
[ ] Callout box untuk informasi penting
[ ] Daftar Isi jika dokumen > 5 halaman
[ ] Versi/tanggal dokumen tercantum
```
